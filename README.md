# ComputerVision

Elegoo Car Project: This is a computer vision project for an Elegoo robot car that uses an ESP32 camera stream for real-time AprilTag detection, pose estimation, and basic autonomous follow control.
It includes:
- Camera calibration tools (intrinsics/distortion estimation)
- MJPEG stream capture and processing pipeline
- AprilTag detection and tracking (`tag36h11`)
- Pose estimation (Python + optional C++ extension for speed)
- Car control modules (keyboard driving + follow behavior)
- Logging and evaluation tools for replay, timing, and performance analysis


LoL_Project: League of Legends minimap champion Detection -- a computer vision project that detects ally and enemy champions on the League of Legends minimap in real time.
The pipeline generates synthetic training data from actual data (labeled in RoboFlow), builds YOLO-format datasets, trains/evaluates a YOLO detector, and runs live inference on the game window with OpenCV overlays.
